# Pong
 A code on C++ to made the legendary game Pong. :ping_pong:
## Controls
Left Paddle:<br>
Up: 'w'<br>
Down: 's'<br>
Right Paddle:<br>
Up: 'i'<br>
Down: 'k'<br>
Quit: 'q'
